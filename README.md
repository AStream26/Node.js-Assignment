# Node.js-Assignment



Some basic Git commands are:

```
$ git clone https://github.com/AStream26/Node.js-Assignment.git
$ cd Node.js-Assignment
$ npm install
```

## Setup

```
 Create .env file that include:

  * dbURL(mongodb url) 
```

## Run the applicatio

```
$ node app.js
```
